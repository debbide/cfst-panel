系统设置

Cloudflare

测速引擎（CloudflareST）

默认已内置官方 CloudflareST（v2.3.5）。启动后会自动解压到数据目录，一般无需手动填写路径。

优先从这个地址拉取别人筛过的候选 IP，再在你本机复测。支持 IP 或 IP#备注 格式。留空才使用本地 IP 文件/全量段。

默认只测 IPv4，适合没有 IPv6 的机器。需要时再手动改成 IPv6 或双栈；IPv6 会用 ipv6.txt 并更新 AAAA 记录。

优选策略

每个域名会同步解析到前 N 个优选 IP（默认 6）。同名同类型多余记录会被清理，降低单 IP 挂掉就全断的风险。

定时任务

支持 5 段 cron：分 时 日 月 周。例如 0 */2 * * * 表示每 2 小时整点执行。
